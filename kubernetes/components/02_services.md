# 2. 서비스 (Services)

## 2.1 서비스 개념

서비스는 파드 집합에 대한 단일 엔드포인트를 제공하는 추상화 계층입니다. 파드의 IP가 동적으로 변경되더라도 안정적인 접근 방법을 제공합니다.

## 2.2 서비스 유형

### 2.2.1 ClusterIP
- 클러스터 내부 통신용
- 기본 서비스 타입
- 내부 IP 할당

### 2.2.2 NodePort
- 외부 접근 가능
- 노드의 특정 포트 개방
- ClusterIP 기능 포함

### 2.2.3 LoadBalancer
- 클라우드 제공자의 로드밸런서 사용
- 외부 접근용
- NodePort 기능 포함

### 2.2.4 ExternalName
- 외부 서비스 매핑
- DNS 이름 제공

## 2.3 서비스 디스커버리

### 2.3.1 환경 변수
- 자동 환경 변수 주입
- 서비스 정보 제공

### 2.3.2 DNS
- 클러스터 DNS
- 서비스 이름으로 접근

## 2.4 로드 밸런싱

### 2.4.1 기본 기능
- 트래픽 분산
- 헬스 체크
- 세션 유지

### 2.4.2 고급 기능
- 가중치 기반 라우팅
- 지역 기반 라우팅
- 카나리 배포 지원
